
# CrossPlatformCodeUtils - A set of classes and functions for cross platform C++ coding

## Current Features
- Works on Windows, macOS, Linux, FreeBSD
- Basic config file
- Basic inf file
- Inter-process synchronized logger
- Thread 
- Mutex
- IPC Named Mutex
- IPC Shared Memory
